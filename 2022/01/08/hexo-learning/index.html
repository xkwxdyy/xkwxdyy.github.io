社交页图标

在主题配置中设置：

  • class: 图标的 css class，主题内置图标详见这里
  • link: 跳转链接
  • tip: 鼠标悬浮在图标上显示的提示文字
  • qrcode: 二维码图片，当使用此字段后，点击不再跳转，而是悬浮二维码
